FastSurfer on Google Colab — Annotated Tutorial

A walkthrough of the **FastSurfer** deep-learning MRI brain-segmentation
pipeline, set up to run end-to-end in Google Colab.

## About this repo
Kept here for my own learning and reference while working with structural
MRI segmentation. See the notebook for the full tutorial.

## What it covers
- FastSurferCNN whole-brain segmentation (95 regions, DKT atlas)
- Local + Docker install options
- Surface reconstruction (recon-surf) and QA tools
